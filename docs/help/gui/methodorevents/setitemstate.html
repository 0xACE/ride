Method 307SetItemState

Applies To:ListViewTreeView

Description

This method is used to set the status of a particular item in a ListView or TreeView object.

The argument to SetItemState is a 2-element array as follows:

[1]Item numberInteger. The index of the item concerned.
[2]StatusInteger

The status of an item is calculated as the sum of one or more of the following state codes:

¯1Error (most likely the Item number is invalid)
1Item has the focus
2Item is selected
8Item is highlighted for dropping
16Item is displayed in bold text
32Item is expanded
64Item is or has been expanded
4096Item is checked. See CheckBoxes
