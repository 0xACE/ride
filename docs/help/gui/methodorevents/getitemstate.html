Method 306GetItemState

Applies To:ListViewTreeView

Description

This method is used to obtain the status of a particular item in a ListView or TreeView object.

The argument for GetItemState is a single item as follows:

[1] Item numberInteger.

Item number is the index of the item concerned. Be aware that this is index original dependent.

The result indicates the state of the item as the sum of one or more of the following codes:

¯1Error (most likely the Item number is invalid)
1Item has the focus
2Item is selected
8Item is highlighted for dropping
16Item is displayed in bold text
32Item is expanded
64Item is or has been expanded
4096Item is checked. See CheckBoxes
