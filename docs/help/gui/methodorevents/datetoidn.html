Method 264DateToIDN

Applies To:CalendarDateTimePickerRoot

Description

This method is used to convert a date from ⎕TS format into an IDN suitable for use in a Calendar object.

The argument to DateToIDN is a 3-element array as follows:

[1]YearInteger
[2]MonthInteger
[3]DayInteger

DateToIDN will also accept a single enclosed argument containing these values. In either case, if you specify more than 3 numbers, excess elements they will be ignored.

Examples

      F.C.DateToIDN 1998 9 11
36048
      F.C.DateToIDN ⊂1998 9 11
36048
      F.C.DateToIDN ⎕TS
36048
      F.C.DateToIDN,⎕TS
36048
