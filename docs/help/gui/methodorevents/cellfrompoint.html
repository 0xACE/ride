Method 200CellFromPoint

Applies To:Grid

Description

This method converts from Grid co-ordinates to cell co-ordinates.

The argument to CellFromPoint is a 2-element array as follows:

[1]y-coordinatenumber in Grid co-ordinates
[2]x-coordinatenumber in Grid co-ordinates

The result is a 2-element vector containing the following:

[1]y-coordinatenumber in cell co-ordinates
[2]x-coordinatenumber in cell co-ordinates
