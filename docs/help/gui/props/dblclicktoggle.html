AppliesÂ To:CoolBar

Description

The DblClickToggle property specifies whether or not the user must single-click or double-click to toggle a child CoolBand between its maximised or minimised state.

DblClickToggle is a single number with the value 0 (single-click toggles state) or 1 (double-click toggles state); the default is 0.
