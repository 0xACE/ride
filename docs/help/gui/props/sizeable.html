PropertySizeable

Applies To

Description

This property determines whether or not an object can be directly resized by the user once it has been created by 竡標C.

It is a single number with the value 0 (the object cannot be resized by the user) or 1 (the object may be resized by the user). The default is 1.

For a Form or SubForm, the Sizeable property may only be set by 竡標C and cannot subsequently be altered using 竡標S. An attempt to do so generates a NONCE ERROR. For a Form, the default value is 1 and the Form occupies a standard resizeable window with a border. Note that the value of Sizeable is independent of the values of the MaxButton and MinButton properties, so that a Form with MaxButton 1 can be maximised even though its Sizeable property is 0.

For other objects, the default value of the Sizeable property is 0. However, setting it to 1 (which may be done dynamically using 竡標S) allows the user to resize it with the mouse.

In all these cases, when the user resizes an object, the object will generate a Configure (31) event.

Sizeable also applies to the Locator object. In this case, a value of 1 implies "rubberbanding" and a value of 0 means "no rubberbanding". See Locator object for further details.
