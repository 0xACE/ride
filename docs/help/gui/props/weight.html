AppliesÂ To:Font

Description

This property specifies the degree of boldness of a font associated with a Font object. It is a number in the range 0 to 1000, where 0 represents very faint and 1000 represents very bold. There is no default; the value of this property reflects the degree of boldness of the font allocated by Windows.

In general, 400 means normal and 700 means bold.
