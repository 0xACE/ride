AppliesÂ To:CalendarDateTimePicker

Description

The HasToday property specifies whether or not the Today date is displayed in the bottom left corner of a Calendar object or in the dop-down calendar of a DateTimePicker.

HasToday is a single number with the value 0 (the date is not shown) or 1 (the date is shown); the default is 1.

See also CircleToday property.
