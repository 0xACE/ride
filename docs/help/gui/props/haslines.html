Description

The HasLines property specifies whether or not tree lines are drawnin a TreeView object. It is a single integer with the value 0, 1 or 2:

0No tree lines
1Tree lines are drawn at all levels except the top level
2Tree lines are drawn at all levels

The user can cause a parent item to expand or collapse by clicking on its corresponding tree line.
