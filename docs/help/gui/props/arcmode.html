PropertyArcMode

AppliesÂ To:CircleEllipse

Description

This property determines how arcs are drawn. Its value is 0, 1 or 2.

0only the arc is drawn
1arcs define "arc segments", with a single straight line joining the two ends of the arc together
2arcs define "pie segments", with lines drawn from the start and end points of the arc to the centre

Note that the segments defined by ArcMode 1 and 2 may be filled (by setting FStyle).
