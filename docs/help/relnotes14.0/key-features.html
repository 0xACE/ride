Key Features

Dyalog APL Version 14.0 provides the following new features, enhancements and changes:

Performance Improvements

Version 14.0 includes a considerable amount of research and development work designed to substantially improve speed of execution. See Performance Improvements .

Language Enhancements

New Language Features

  • New terminology is introduced in Version 14.0 to describe sub-arrays of an array as cells. This terminology is used to describe the workings of the new Rank (⍤) and Key (⌸) operators, the function Tally (≢), and the extension to dyadic Iota (⍳). See Cells and Sub-arrays.
  • New Rank operator ⍤. See Monadic Examples
  • New Key operator ⌸. See Key Operator.
  • New function Tally ≢. See Tally
  • Function Trains. See Function Trains.

New I-Beam Features

  • A function is provided to support WPF data binding. See Data Binding.
  • A function is provided to perform fast index-of operations on inverted table structures. See Inverted Table Index Of.
  • A function is provided to obtain the current type of an array without it being squeezed first. See Unsqueezed Type.
  • A function is provided to compress and decompress vectors of short integers. See Compress Vector of Short Integers.
  • A function is provided to serialise and de-serialise an array. See Serialise/Deserialise Array.
  • A function is provided to update the function timestanp and user. See Update Function Time Stamp.
  • A function is provided to affect fine control over memory allocated to the workspace. See Specify Workspace Available.
  • A function is provided to flush the Session caption. See Flush Session Caption.
  • A function is provided to close all windows. See Close All Windows.
  • A function is provided to set workspace save options. See Set Workspace Save Options:.

Extensions

  • Dyadic ⍳ has been extended to matrices and higher-rank arrays. See Index Of .
  • Mix (↑⍵ if ⎕ml<2; ⊃⍵ if ⎕ml≥2) has been extended. See Mix .
  • The APL2-compatible version of Mix (⊃⍵ when ⎕ml≥2) has been extended. See Mix With Axis.
  • Monadic operators may now be named and assigned. See Rationalisation of Monadic Operators .
  • Dyadic operators may now be bound with their right arguments to form monadic operators. See Right Operand Currying .
  • Options for ⎕XML may now be specified using the Variant operator ⍠. This becomes the recommended approach, although the use of the optional left argument will be retained for backwards compatibility. See Example.
  • Roll can now generate random numbers in the range 0 - 1. See Roll.
  • Table (monadic ⍪) may now be used in selective assignment.
  • ⎕RL has been extended to initialise the random number generator with a random seed. See Random Link Extension.

Component File System Improvements

  • Data in component files may now be compressed to improve performance when file access is slow. See Compressed Components.
  • File properties may now be set when the file is created using a function derived from ⎕FCREATE and the Variant operator ⍠. See File Create and Variant.
  • ⎕FREAD can now read multiple components. See File Read.
  • Options for ⎕FCHK may now be specified using the Variant operator (recommended) instead of using its optional left argument. See File Check and Variant.

IDE Enhancements

  • The system now generates a SessionPrint event when a value is about to be displayed in the Session. This event is used by the two new user commands described below. For further details, see SessionPrint.
  • ]boxing user command optionally causes arrays displayed in the Session to be drawn in boxes to display their structure. See ]boxing.
  • ]rows user command controls how rows in multi-row output to the Session are displayed. See ]rows.
  • New Editor feature to align comments. See Aligning Comments.
  • New Editor toolbar buttons. See New Editor Toolbar Buttons.
  • New Editor options Allow search to wrap and Skip blank lines when tracing. See New Editor Options.
  • New Tracer extension to allow the Editor to be called when the cursor is on any whitespace.
  • New Editor option Remember previous window position (Classic Mode only)
  • New Tracer option Limit tracer display to function in script.
  • Facility to customise the window captions for the IDE. See Window Captions .
  • New function Set Workspace Save Options determines whether or not the system will reset Trace, Stop and Monitor settings whenever a workspace is saved. See Set Workspace Save Options:.
  • Value tips can now be used to investigate the syntax of external functions.

.Net Interface

  • Under a licensing agreement with Syncfusion, Dyalog includes the Syncfusion library of WPF controls. These may be used by Dyalog APL users to develop applications, and may be distributed with Dyalog APL run-time applications.The Syncfusion libraries comprise a set of .NET assemblies which are supplied in the Syncfusion/4.5 sub-directory of the main Dyalog APL installation directory (for example: c:\Program Files\Dyalog\Dyalog APL-64 14.0 Unicode\Syncfusion\4.5.. See Syncfusion Libraries.
  • Version 14.0 includes support for Data Binding. See Example 1.
  • The .Net interface includes a new DLL. The dyalogdata4.5.dll provides advanced support for Data Binding and Syncfusion. In particular it provides the INotifyCollectionChanged interface which is required to support data binding of collections and lists. This DLL requires .Net Version 4.5 and is not used unless .Net 4.5 is enabled.
  • It is now possible to specify overloads and casts for calling .Net functions.

APL Applications as Windows Services

Features to allow the implementation of APL applications as Windows Services are now integrated into the Dyalog system.
