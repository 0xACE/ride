Dyalog APL 14.0 Help

Welcome to Dyalog APL's online Help and Documentation for Version 14.0.

Documentation is available in two formats: Help systems and PDF files.

Help Systems:

  • If you are using Dyalog APL under Windows, then the documentation is referenced under the Help Menuitem in the session. Note also that hitting <F1> when the cursor is over a primitive function or operator or a system function etc. will bring up an appropriate page in the help system.

PDF Files:

  • All of the documentation supplied with a release of Dyalog APL resides in the help subdirectory of the Dyalog installation.
  • If you wish to print any of the manuals which form part of the Help system, then PDFs suitable for printing are available at http://docs.dyalog.com/14.0/; the PDFs supplied with the release are intended for on-screen browsing.

The documentation is updated throughout the life of each version of Dyalog; the latest versions of the files and the online help for all supported versions are available at http://docs.dyalog.com and http://help.dyalog.com

If you report an issue with the documentation, could you please include the following information: Version 14.0, revision 20140930.
