Round Parentheses

Round brackets are used in APL expressions to control the order of execution. A right parenthesis is also used to introduce a System Command.

Square Brackets

Square brackets are have two distinct meanings in APL. They are used to select sub-arrays from an n-dimensional array and to indicate an axis along which a function (operand) is to apply. See Bracket Indexing and Bracket Axis Operator.

Curly Brackets

Curly brackets have two distinct meanings in Dyalog APL. They are used to indicate shy results and optional left arguments in defined functions. They are also used to enclose the definition of a dynamic function. See Model Syntax and D-functions.
