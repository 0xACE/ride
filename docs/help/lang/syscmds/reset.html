)RESETReset State Indicator

This command cancels all suspensions recorded in the state indicator and discards any unprocessed events in the event queue.

)RESET also performs an internal re-organisation of the workspace and process memory. See Workspace Available for details.

Example

      )SI
#.FOO[1]*
⍎
#.FOO[1]*
 
      )RESET
 
      )SI
