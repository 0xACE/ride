Default Left Argument

The special  syntax: ⍺←expr is used to give a default value to the left argument if a Dynamic Function is called monadically. For example:

      root←{      ⍝ ⍺th root
          ⍺←2     ⍝ default to sqrt
          ⍵*÷⍺
      }

The expression to the right of  ⍺← is evaluated only if its Dynamic Function is called with no left argument.
