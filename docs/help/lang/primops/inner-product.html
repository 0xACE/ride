R←Xf.gYInner Product

f must be a dyadic function.  g may be any dyadic function which returns a result.  The last axis of X must have the same length as the first axis of Y.

The result of the derived function has shape (¯1↓⍴X),1↓⍴Y.  Each item of R is the result of f/xg¨y where x and y are typical vectors taken from all the combinations of vectors along the last axis of X and the first axis of Y respectively.

Function f (and the derived function) need not return a result in the exceptional case when 2=¯1↑⍴X.  In all other cases, function f must return a result.

If the result of xg¨y is empty, for any x and y, a DOMAIN ERROR will be reported unless function f is a primitive scalar dyadic function with an identity element shown in Identity Elements.
