Axis Specification

Some operators may include an axis specification.  Axis is itself an operator.  However the effect of axis is described for each operator where its specification is permitted. ⎕IO is an implicit argument of the function derived from the Axis operator.

The description for each operator follows in alphabetical sequence.  The valence of the derived function is specifically identified to the right of the heading block.
