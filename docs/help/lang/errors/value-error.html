6VALUE ERROR

This report is given when either:

  • There is no active definition for a name encountered in an expression.
  • A function does not return a result in a context where a result is required.

Examples

     X
VALUE ERROR
     X
     ^
 
     ∇ HELLO
[1]    'HI THERE'
[2]  ∇
 
     2+HELLO
HI THERE
