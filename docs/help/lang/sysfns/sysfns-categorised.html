System Functions Categorised

System functions, variables and constants provide various services related to both the APL and the external environment. They have distinguished names beginning with the ⎕ symbol and are implicitly available in a clear workspace.

The following tables identify system functions, variables and constants divided into relevant categories.  Each function is described in alphabetical order in this chapter
