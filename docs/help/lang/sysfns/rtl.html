⎕RTLResponse Time Limit

A non-zero value in ⎕RTL places a time limit, in seconds, for input requested via ⍞ , ⎕ARBIN , and ⎕SR.  ⎕RTL may be assigned any integer in the range 0 to 32767.  The value is a clear workspace is 0.

Example

      ⎕RTL←5 ⋄ ⍞←'FUEL QUANTITY?' ⋄ R←⍞
FUEL QUANTITY?
