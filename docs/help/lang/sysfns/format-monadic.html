R←⎕FMT YFormat (Monadic)

Y may be any array.  R is a simple character matrix which appears the same as the default display of Y.  If Y contains control characters from ⎕TC, they will be resolved.

Examples

      A←⎕FMT '∩' ,⎕TC[1],'∘'
 
      ⍴A
