R←X⊢YRight

X and Y may be any arrays. The result R is the right argument Y.

Example

      42 ⊢'abc' 1 2 3
 abc  1 2 3

Note that when ⊢ is applied using reduction, the derived function selects the last sub-array of the array along the specified dimension. This is implemented as an idiom.
