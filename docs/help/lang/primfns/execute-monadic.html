R←⍎YExecute (Monadic)

Y must be a simple character scalar or vector.  If Y is an empty vector, it is treated as an empty character vector.  Y is taken to be an APL statement to be executed.  R is the result of the last-executed expression.  If the expression has no value, then ⍎Y has no value.  If Y is an empty vector or a vector containing only blanks, then ⍎Y has no value.

If Y contains a branch expression which evaluates to a non-empty result, R does not yield a result.  Instead, the branch is effected in the environment from which the Execute was invoked.
