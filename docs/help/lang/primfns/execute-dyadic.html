R←X⍎YExecute (Dyadic)

Y must be a simple character scalar or vector.  If Y is an empty vector, it is treated as an empty character vector.  X must be a namespace reference or a simple character scalar or vector representing the name of a namespace.  Y is then taken to be an APL statement to be executed in namespace X.  R is the result of the last-executed expression.  If the expression has no value, then X⍎Y has no value.

Example

      ⎕SE ⍎ '⎕NL 9'
